## API Details:

**Endpoint**: `veo-3.1-generate-preview`
**Project**: `vibers-paris-ai-hackathon`
**Region**: `us-central1`

**Parameters**:
- `aspectRatio`: `16:9` (widescreen)
- `durationSeconds`: `8` (max for now)
- `resolution`: `720p`
- `personGeneration`: `allow_all`
- `generateAudio`: `false` (we use TTS instead)

---

## Troubleshooting:

**"Veo generation failed"**
- Check Google Cloud credentials
- Verify Vertex AI API is enabled
- Check photo is valid JPEG
- Ensure project ID is correct

**"Taking too long"**
- Veo can take 1-2 minutes
- Be patient, it's generating a full AI video!
- Check operation status in console

**"Video quality low"**
- Use higher resolution photo
- Ensure good lighting in photo
- Try different angles/expressions
